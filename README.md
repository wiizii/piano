# piano
자바스크립트로만 구현한 피아노.
어떠한 음원파일 없이 AudioContext Web API를 이용하여 구현

# docs
https://developer.mozilla.org/ko/docs/Web/API/AudioContext

# description
음원의 파형을 직접 만든 후에 음색과 볼륨을 조절하여 음(note) 생성
